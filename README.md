A sleek, high-intensity web application designed to test knowledge on Indian history, geography, technology, and culture. This project features a "Lightning Round" timer, dynamic background transitions, and a comprehensive result review system.

*Features*
Lightning Round Logic: Each question must be answered within 5 seconds.
Visual Pressure Cues: The webpage background changes color (Green → Yellow → Red) as the timer runs out.
Dynamic Data Fetching: Simulated API fetch architecture for a modern, scalable approach.
Premium UI: Includes a real-time progress bar, responsive button grid, and smooth CSS transitions.
Post-Quiz Analysis: Displays a full answer key at the end of the session so users can learn from their mistakes.

*Tech Stack*
HTML5: Semantic structure for the quiz interface.
CSS3: Custom styling featuring Flexbox, Grid, and color transitions.
JavaScript (ES6+): Async/Await for data fetching, DOM manipulation, and interval-based timer logic.

*How to Run*
Clone or Download: Save the project files (index.html, style.css, script.js) into one folder.
Open in Browser: Simply double-click index.html.
(Recommended: Use the Live Server extension in VS Code for a better development experience.)
Play: Click "Start Quiz" and try to beat the clock!
